## About
* **Konsole** as the terminal.
* **Catppuccin Macchiato** with Lavender accents as the color scheme.
* **Kwin + Bismuth** as the window manager.
* **Rofi** as the application launcher.
* **Polybar** as the main bar.
* **SDDM** as the login manager
